The most important parts of Virtual Qmail package are presented here.
  • Virtual Qmail represents to core of a system. It's a set of programs (and libraries) allowing to manage system from CLUE.
  • Vqwww is the framework for allowing access to the vq system via WWW pages (using CGI).
  • Freemail is a set of modules for vqwww and additional programs allowing to create site hosting free e-mail accounts.
  • Hosting is a set of programs and modules for vqwww that can be used to create hosting system (manage users, WWW aliases, FTP access, etc.)
