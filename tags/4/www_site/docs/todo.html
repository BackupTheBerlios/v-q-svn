Following is a list of things that will be implemented in next version:

  • Manager of -ext addresses.
  • Fix quota support for domains (also copying default values)
  • Rewriting of daemon's code - it will be separated into program (for example authorization module for PostgreSQL will be load using dynamic linking).
  • Precompiled binary packages for OpenBSD and Slackware.

Things that will be implemented in following versions:

  • WWW manager of all domains under vq.
  • Oracle support (maybe ODBC support).
  • Postfix support.
  • Database connections poll (programs will open n connections at start and use them in the same time - threads support).
