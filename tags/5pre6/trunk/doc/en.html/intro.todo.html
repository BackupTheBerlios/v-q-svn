1.3. Planned features

This is not a closed list. If you have any suggestions please contact Pawe³ Niewiadomski.

Planned features are:

  • quota support

  • i18n support

  • manager of extended addresses known from Qmail

  • WWW interface for administrators

  • Postfix and Sendmail support.

  • remote configuration - another CORBA interface allowing to configure system
