File: autoresp_ttl

Mode of files created in qmail's home. 0 means octal form.
File: split_dom
Level of splits of domain name. Directory "domains" includes domains registered in system. It's possible to lower number of entries per directory by creating tree. If you leave this with value 3, path for "test.org.pl" in "domains" will look: domains/pl/org/test/test.org.pl. If you don't want any splits set it to 0.
File: split_user
Level of splits of user name. To lower number of entries in directory containing user mailboxes tree is created. For value 3 entries in domain's directory look as following: for user test, path is t/e/s/test/.
